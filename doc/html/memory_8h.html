Compares two blocks of memory.

Compares the first n bytes of of b1 and b2.

Parameters
b1First block of memory to compare with.
b2Second block of memery to compare with.
nNumber of bytes of each block to compare.
Return values
void*Status of compared memory
  • <0 first non equal byte has a lower value in b1
  • 0 both blocks of memory are equal
  • >0 first non equal byte has a lower value in b2
